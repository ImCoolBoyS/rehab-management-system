# 精神康复中心管理系统 — Logo 设计需求文档

## 项目背景

一套面向社区精神康复服务点的业务管理系统，覆盖学员建档、康复训练、入户探访、评估管理等全流程。使用者为市级残联管理人员、乡镇站点主管、一线社工。

## 品牌定位

| 维度 | 描述 |
|------|------|
| 行业 | 医疗卫生 / 精神康复 / 社区服务 |
| 调性 | 专业、温暖、可信赖、有希望感 |
| 受众 | 残联管理人员、社工、康复学员家属 |
| 关键词 | 康复、关怀、社区、新生、希望 |

## 设计方向

### 方向 A：抽象图形为主
- 不需要具象的"大脑"或"十字"
- 优先用几何图形组合表达"呵护"、"生长"、"联结"
- 参考风格：简约几何、双色渐变、留白多

### 方向 B：文字为主 + 辅助图形
- 以中文"康"或"复"字为核心做图形化设计
- 结合柔和弧线或正负形手法
- 适合偏向行政/严肃的场景

## 色彩参考

| 色系 | 用途 | 色值参考 |
|------|------|---------|
| 青绿色 | 主色，医疗专业感 | `#0d9488` ~ `#14b8a6` |
| 暖金色 | 辅助色，希望/温暖 | `#d97706` ~ `#f59e0b` |
| 深灰蓝 | 文字/背景 | `#0f172a` ~ `#1e293b` |

## 应用场景

| 场景 | 尺寸 | 说明 |
|------|------|------|
| 侧边栏顶部 | 40×40px | 主页导航，深色背景上使用 |
| 登录页 | 80×80px | 居中展示，浅色背景上使用 |
| Favicon | 32×32px | 浏览器标签图标 |
| 系统标题左侧 | 24×24px | 文字前的小标 |

## 设计禁忌

- 不要使用红/粉色主色调（易与精神科负面印象关联）
- 不要使用过于复杂的渐变或立体效果
- 不要出现"精神病"字样
- 不要使用医疗十字标志
- 避免过于幼圆或卡通风格

## 交付格式

- SVG 源文件（支持深色/浅色两种背景）
- PNG 导出（各应用场景尺寸）
- 简洁版本（纯图标，16×16起）
- 完整版本（图标+文字组合）
